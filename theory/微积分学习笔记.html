1. 什么是微积分？

微积分是数学的一个重要分支，主要研究变化和累积。它由两部分组成：

🎯 微积分核心思想可视化

▶ 微分学 ▼

微分学研究的是变化率的问题。

比如：

  • 汽车的速度是如何变化的？
  • 曲线的斜率是多少？
  • 函数在某些点的瞬时变化率？

核心概念：导数（Derivative）

▶ 积分学 ▼

积分学研究的是累积的问题。

比如：

  • 曲线下的面积是多少？
  • 物体移动的总距离是多少？
  • 函数在一定区间内的累积效应？

核心概念：积分（Integral）

💡 小贴士：微积分的核心思想可以用一句话概括："微分是把整体分解为局部，积分是把局部累积为整体。"

2. 函数定义

函数是微积分的基础概念。理解函数是学习微积分的第一步。

📈 交互式函数绘图器

2.1 什么是函数？

函数（Function）是一种特殊的对应关系：对于每一个输入值，都有唯一确定的输出值与之对应。

y = f(x)
x 是自变量，y 是因变量，f 是函数关系

2.2 函数的三要素

  1. 定义域：自变量 x 的取值范围
  2. 对应法则：f 的具体表达式
  3. 值域：因变量 y 的取值范围

📝 例子：一次函数

f(x) = 2x + 3

• 定义域：x ∈ R（所有实数）

• 对应法则：每个 x 乘以 2 再加 3

• 当 x = 1 时，f(1) = 2×1 + 3 = 5

▶ 常见函数类型 ▼

1. 一次函数：f(x) = kx + b

2. 二次函数：f(x) = ax² + bx + c

3. 指数函数：f(x) = aˣ

4. 对数函数：f(x) = logₐx

5. 三角函数：f(x) = sin x, cos x, tan x

3. 极限概念

极限是微积分的基石。它描述的是当自变量趋近于某个值时，函数值的变化趋势。

🎯 极限可视化：sin(x)/x 当 x→0

x = 1.0, sin(x)/x = 0.8415

3.1 极限的直观理解

想象你走在一条路上，目标是到达某个点。当你离这个目标点越来越近时，你的位置就"趋近于"那个点。

lim(x→a) f(x) = L
当 x 趋近于 a 时，f(x) 趋近于 L

3.2 重要极限公式

lim(x→0) sin(x)/x = 1
⚠️ 注意：极限存在的前提是左极限和右极限都存在且相等。

📝 例子：

计算 lim(x→2) (x² - 4)/(x - 2)

解：

lim(x→2) (x² - 4)/(x - 2) = lim(x→2) (x-2)(x+2)/(x-2) = lim(x→2) (x+2) = 4

4. 导数入门

导数是函数的瞬时变化率，几何意义是曲线在某点的切线斜率。

📐 导数几何意义：切线斜率

x₀ = 1.0, f'(x₀) = 2.0

4.1 导数的定义

f'(x) = lim(Δx→0) [f(x+Δx) - f(x)] / Δx

4.2 基本导数公式

▶ 点击查看常用导数公式 ▼

1. 常数函数：(C)' = 0

2. 幂函数：(xⁿ)' = nxⁿ⁻¹

3. 指数函数：(eˣ)' = eˣ

4. 对数函数：(ln x)' = 1/x

5. 三角函数：

• (sin x)' = cos x

• (cos x)' = -sin x

• (tan x)' = sec² x

4.3 导数的几何意义

导数 f'(x₀) 表示函数 f(x) 在点 x₀ 处的切线斜率。

📝 例子：

求 f(x) = x² 在 x = 3 处的导数

解：

f'(x) = 2x

f'(3) = 2×3 = 6

所以，曲线 y = x² 在点 (3, 9) 处的切线斜率为 6。

5. 积分入门

积分是导数的逆运算，主要用来计算面积、体积等累积量。

📏 定积分几何意义：曲线下的面积

-2 2

5.1 不定积分

不定积分是求原函数：

∫ f(x) dx = F(x) + C
其中 F'(x) = f(x)，C 是积分常数

5.2 定积分

定积分计算函数曲线下的面积：

∫[a→b] f(x) dx = F(b) - F(a)
牛顿-莱布尼茨公式

5.3 基本积分公式

▶ 点击查看常用积分公式 ▼

1. ∫ xⁿ dx = (xⁿ⁺¹)/(n+1) + C (n ≠ -1)

2. ∫ (1/x) dx = ln|x| + C

3. ∫ eˣ dx = eˣ + C

4. ∫ sin x dx = -cos x + C

5. ∫ cos x dx = sin x + C

📝 例子：

计算 ∫[0→2] x² dx

解：

∫ x² dx = (x³)/3 + C

∫[0→2] x² dx = (2³)/3 - (0³)/3 = 8/3
