⚙️ PI控制理论 - 交互式学习

提升机控制系统核心原理

📚 什么是PI控制？

PI控制是比例（Proportional）和积分（Integral）控制的组合，是工业控制中最常用的控制策略之一。

对于提升机这样的慢速、大惯性系统，PI控制已经足够实现精确、平稳的控制，不需要微分（D）控制。

PI控制器输出公式：
Output = Kp × error + Ki × ∫error dt

其中：
• Kp = 比例增益（响应速度）
• Ki = 积分增益（消除稳态误差）
• error = 目标值 - 实际值

💡 为什么提升机不用D控制？

  • 慢速系统：提升机转速、位置变化缓慢，误差变化率小，D项输出几乎为0
  • 大惯性：系统响应慢，D项的"预测"作用无法发挥
  • 副作用：D项会引入噪声，导致系统震荡，影响平稳性
  • PI已足够：P项快速响应，I项消除稳态误差，二者配合已可实现精确控制

1️⃣ 比例（P）控制 - 快速响应

核心原理

比例项输出与当前误差成正比：误差越大，输出越大。

P = Kp × error

其中：error = 目标值 - 实际值

直观理解：开车油门

🚗 类比：保持定速行驶

场景：你想保持60km/h的速度行驶

  • 当前速度55km/h → 误差5km/h → P项输出中等油门
  • 当前速度40km/h → 误差20km/h → P项输出大油门
  • 当前速度60km/h → 误差0 → P项输出0油门

问题：上坡时，P项输出0油门，但重力会让车减速，最终稳定在58km/h（稳态误差）！

P控制的缺陷：稳态误差

⚠️ 稳态误差问题

现象：系统稳定后，实际值永远达不到目标值，存在一个固定误差。

原因：当误差为0时，P项输出为0，但系统存在自发偏移力（如摩擦力、重力），需要持续的调控力来抵消。

解决方案：加入积分（I）项！

2️⃣ 积分（I）控制 - 消除稳态误差

核心原理

积分项输出与历史所有误差的累积成正比：只要存在误差，积分项就会持续累积，直到误差为0。

I = Ki × ∫error dt

离散形式：I = Ki × (error₁ + error₂ + ... + errorₙ) × Δt

直观理解：蓄水池水位控制

💧 类比：保持目标水位

场景：你想将蓄水池水位保持在1米

  • 纯P控制：水位0.99米时，P项输出小功率，但池子漏水，最终稳定在0.99米（稳态误差）
  • 加入I控制：每次水位不足都会被"记下来"，积分项逐步加大水泵功率，直到水位达到1米
  • 关键：即使误差很小（如0.001米），只要不为0，积分项就会持续累积，最终消除误差

3️⃣ 提升机中的PI控制应用

提升机的三闭环控制

提升机采用三闭环控制系统，其中位置和速度控制都使用PI控制：

控制环 控制器类型 作用 实现位置
位置闭环 比例（P）控制器 使实际位置精确跟随设定位置 FM458
速度闭环 比例积分（PI）控制器 使实际速度精确跟随设定速度，消除稳态误差 FM458
电流闭环 PI控制器 控制电机转矩电流 传动装置

速度PI控制器详解

速度控制器输出（力矩设定值）：
Output = Kp × (速度误差) + Ki × ∫(速度误差) dt

其中：
• 速度误差 = 设定速度 - 实际速度
• 设定速度 = 设定点发生器输出 + 位置控制器输出
• 实际速度 = 速度编码器（编码器1）测得的速度

为什么速度控制需要I项？

🏗️ 类比：提升机速度控制

场景：提升机以2m/s的速度上升，负载50吨

  • 纯P控制：速度达到2m/s时，P项输出为0，但负载重力会让提升机减速，最终稳定在1.95m/s（稳态误差）
  • 加入I控制：每次速度不足都会被累积，I项输出逐步加大力矩，直到速度稳定在2m/s
  • 关键：I项输出的力矩正好抵消负载重力，保持恒速运行

⚠️ 观察要点

  • Ki太小：稳态误差消除慢，速度恢复时间长
  • Ki太大：积分累积过快，速度可能超调，导致震荡

📊 PI参数调优技巧

分步调参法

  1. 先调P：将Ki设为0，逐渐增大Kp，直到系统响应较快且有轻微超调
  2. 再调I：保持Kp不变，逐渐增大Ki，直到稳态误差在可接受时间内消除
  3. 微调：根据实际情况微调Kp和Ki，平衡响应速度和稳定性

提升机PI控制特殊考虑

✅ 最佳实践

  • 加速度预控：在PI控制基础上，加入加速度预控项，补偿惯性力
  • 载荷预控：根据装载量，提前给出相应的力矩预控
  • 机械预控：补偿两个箕斗的自重差和首尾绳重量差
  • 平滑设定值：使用加速度变化率控制，让设定值曲线全程平滑可导

🎓 学习总结

✅ 核心要点

  1. P控制：快速响应误差，但存在稳态误差
  2. I控制：消除稳态误差，但可能引入超调
  3. PI控制：P快速响应 + I消除稳态误差，工业最常用
  4. 提升机不用D：慢速大惯性系统，D项无法发挥作用，反而有副作用
  5. 三闭环：位置P + 速度PI + 电流PI，实现精确控制
